Implementasi Firebase Firestore pada Aplikasi Cepu

1. install dependensi: cloud_firestore, image_picker,
   geolocator, url_launcher
2. Buat model post.dart di folder models, dengan properti:
   id, image, description, category, latitude, logitude, user_id, user_fullname
3. Buat service post_service.dart di folder service
4. Buat screen : add_post_screen.dart dan detail_screen.dart
